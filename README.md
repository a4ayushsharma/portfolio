# Personal Portfolio  
A modern, interactive single-page website to showcase who I am, what I do, and the projects I’ve built.

## About the Project  
This repository hosts my personal portfolio website — a digital space where I present my skills, experience, and creative journey as a tech-enthusiast and developer.  
Designed and developed with a focus on simplicity, interactivity, and modern design, this portfolio is structured to highlight:  
- Who I am & what I can offer  
- My technical skills and tools  
- Selected projects and work  
- My contact / social links  

## Built With  
- HTML5  
- CSS3  
- JavaScript  

## Features  
- Clean and minimal UI / UX  
- Interactive navigation (scrolling, anchored sections)  
- Responsive layout — works on desktops, tablets & mobile devices  
- Sections for About, Skills, Projects & Contact
